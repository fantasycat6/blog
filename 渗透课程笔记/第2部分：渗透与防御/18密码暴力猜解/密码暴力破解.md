# python实现暴力破解

## python暴破 

1. 从字典读取值，生成密码 
2. HTTP连接到需要暴破的地址 
3. 获得HTTP响应，分析响应结果，看看有没有错误提示 “Username and/or password incorrect.” 
4. 如果有提示，就继续下一次循环 
5. 如果没有，就代表暴破成功
