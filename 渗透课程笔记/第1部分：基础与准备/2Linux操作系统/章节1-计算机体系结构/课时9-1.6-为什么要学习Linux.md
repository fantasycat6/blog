# 为什么要学习Linux?

1、使用网络的程序，都有一个服务器;
2、服务器，大部分使用的是Linux操作系统。 <a name="EUM1M"></a>

# 学习方法

1、先学习重点
2、分门门别类地学
3、不要强行记命令

    cd change directory
    pwd print working directory
    rm remove
    cp copy

    -f force 强制
    -v verbose详细
    -r recursive递归
    	 reverse倒序
    -i inquir询问

4、多用，自然熟练
