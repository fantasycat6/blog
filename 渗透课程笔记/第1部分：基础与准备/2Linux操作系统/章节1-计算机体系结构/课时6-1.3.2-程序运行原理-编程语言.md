# 编程语言

<a name="g3Bys"></a>

## 第一代语言：

机器语言:二进制代码语言
01000000101 <a name="DIpzv"></a>

## 第二代语言：

汇编语言:用英文字母或符号串来替代操作
LOAD A
ADD B
STORE C <a name="nmm2t"></a>

## 第三代语言：

面向过程语言：Fortran，Basic，C，Pascal
面向对象语言：SmallTalk，EIFFEL，Ada，C++，c#，python，Java，golang
高级编程语言是面向过程和面向对象语言。贴近自然语言，不直接操作硬件，可移植
C= A+ B;
